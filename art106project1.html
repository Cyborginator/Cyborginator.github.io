Project 1: LED String

For this project we made a string of LEDS in series hooked up to a battery with some resistors to keep them from burning out and then made a second one which we wired up in parallel.

I followed the diagram below to create my LED string being careful to wire up the positive and negative leads correctly.

After wiring like the diagram we added a makeshift switch using folded foam and aluminum foil. Then we were instructed to find a way to put it to use and take a video, so I thought a good example use case for it is with my trash can where you put your foot down on both the dwitch and lever and it opens the trash and lights up the garbage can.
